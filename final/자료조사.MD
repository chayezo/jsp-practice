### 로그인 / 회원가입
1. 로그인
   - 모달창* { 전체적으로 많이 사용 }
   - 이메일로 간편 로그인
     - 가입되지 않은 이메일이면 알려줌.
   - facebook으로 로그인 -> { 페이스북은 구현 까다로우니까 카카오/네이버로 변경 }
   - 비밀번호 재설정
     - 이메일 이용해서 진행 : 비밀번호 변경 이메일 보내줌

2. 회원가입
   - 이름, 이메일, 비밀번호 이용한 간편가입
   - 언어선택
